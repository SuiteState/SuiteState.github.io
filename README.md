# SuiteState — Official website

This repository hosts the source for [suitestate.com](https://suitestate.com),
the public-facing site for the **SuiteState** brand: open-source Odoo modules
and consultancy from Dubai.

## Structure

- `index.html` — single-page landing
- `style.css` — all styles (no framework, no build step)
- `CNAME` — custom domain configuration for GitHub Pages

## Editing

This is a static site. Edit the HTML or CSS directly, commit, and push —
GitHub Pages rebuilds automatically (typically within 1 minute).
